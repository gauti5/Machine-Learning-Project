# Machine-Learning-Project



Creating Conda Environment

'''
conda create -p venv python==3.11.5 -y
'''

Activating conda
'''
conda activate venv/ 
'''
or
'''
conda activate venv
'''

Installing requirements.txt
'''
pip install -r requirements.txt
'''
